# Square Gardener

A React-based web application for managing square foot gardens. Track plants, watering schedules, and harvest times with an intuitive interface.

## Live Demo

**[View Live Site](https://xmerr.github.io/Square-Gardener/)**

## Tech Stack

- React 19 + Vite 7
- React Router DOM 7
- Tailwind CSS
- Vitest + React Testing Library

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Xmerr/Square-Gardener.git
cd Square-Gardener

# Install dependencies
npm install
```

### Running Locally

```bash
# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Building for Production

```bash
npm run build
```

## Testing

This project enforces 100% code coverage.

```bash
# Run tests in watch mode (during development)
npm test

# Run tests with coverage report
npm run test:ci

# Run tests with UI
npm run test:ui
```

## Linting

```bash
npm run lint
```
